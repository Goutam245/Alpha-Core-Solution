# Alpha Core Solutions

Professional security solutions website for the Kingdom of Saudi Arabia's trusted security partner.

## 🌟 Overview

A modern, bilingual (English/Arabic) website showcasing comprehensive physical security solutions including access control, CCTV surveillance, metal detection, and X-ray screening systems.

## ✨ Features

- 🌐 Bilingual support (English/Arabic)
- 📱 Fully responsive design
- 🎨 Modern, professional UI
- 🎥 Video background hero section
- 🍪 GDPR-compliant cookie consent
- ⚡ Fast loading & optimized performance

## 🛠️ Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- React Router

## 🚀 Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📦 Deployment

Deploy the `dist/` folder to any static hosting service:
- Vercel
- Netlify
- GitHub Pages
- AWS S3
